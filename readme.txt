文件夹说明

1、C_RC 网上下载的一个SN8P2501D芯片C语言示例代码

2、HU  弧形形加热板资料原理图PCB 源程序都在文件夹内

3、U   U形加热板资料原理图PCB 源程序都在文件夹内

4、test  U形加热板C语言源程序适配IC SN8P2501D 后来因为C语言IDESN8 C Studio_V157(583.297)不支持仿真器仿真，
     所以保留了下来，但U形板和弧形板的汇编代码都是按这个思路编写的。

5、带定时功能 这个是另一个版本客户要求增加一个软定时器功能的加热板。目前用的是可调电位器调时间最长8小时定时时长，
    每半个小时定时为1个档共15档，0档为关闭定时功能。

6、按键板.png、红外摇控板说明书.jpg 这两张图是红外摇控器功能说明，这个摇控器是配套U形和弧形加热板使用。
